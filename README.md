# Wedding Invitation - Mohanraj & Janaki Priya

A beautiful, animated wedding invitation website built with React, Framer Motion, and Material UI.

## Features

- 🎨 Elegant, responsive design
- ✨ Smooth animations with Framer Motion
- 📱 Mobile-friendly layout
- ⏳ Countdown timer
- 🗺️ Venue map integration

## Tech Stack

- React
- Vite
- Framer Motion
- Material UI (MUI)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
